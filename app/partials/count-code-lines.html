Kata13: Count Java code lines

Philosophy: we only want to count the lines and do not need to use complex regex to match the comments.
  1. Delete comments while ignoring /* that are inside double quotes (as long as double quote is not inside a comment)
  2. Split the input into an array of lines
  3. Loop through each line and strip white space. If the first character of the line is not //, add 1 to counter
Unit tests can be run using "npm test"
